Welcome to the 0.9.1 Release

This 0.9.1 release contains a few bug fixes and adds the ability to expose static WSDL, with the option for transforming location based on WSDL request URL. See Status for release notes.

The JiBX/WS project has built on, and replaced, the jibxsoap project, adding enhanced SOAP support as well as POX, TCP, and XBIS support.

The docs subdirectory contains the full JiBX/WS documentation.
